# resume.md 格式

用户粘贴的简历文本的落盘底本，是 Interrogator 所有 Probe Point 的**唯一取材来源**，
也是跨会话的稳定引用对象。

## 来源

纯文本粘贴。本 skill 不解析 PDF / Word —— 让用户直接粘贴简历内容，原样整理落盘。

## 结构

把粘贴内容按以下分块整理（缺失的块可省略）：

```md
# 简历底本

## 个人概要
{一句话定位、年限}

## 工作 / 项目经历
### {公司 / 项目名}（{时间段}）
- {职责 / 成果，尽量保留原文措辞}

## 技能
- {技能点，逐条列出，便于标注 Probe Point}

## 教育 / 其他
{学历、证书、开源等}
```

## 规则

- **忠实落盘**：保留用户原始措辞，不要替用户美化或编造。
- 整理时顺手标出初步的 Probe Point（高含金量或可疑条目），供拷问循环使用。
- 这是底本——后续简历修改建议针对它提出，但建议写在 Debrief 里，不直接覆盖本文件
  （除非用户明确要求采纳）。
